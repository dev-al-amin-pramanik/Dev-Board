Question-1: What are the different ways to select an element in the DOM?

The different ways to select an element in the DOM are:

1. getElementById("id")
2. getElementsByClassName("class")
3. getElementsByTagName("tag")
4. getElementsByName("name")
5. querySelector("selector")
6. querySelectorAll("selector")

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML allows us to add or replace HTML inside an item, which means it may include tags. innerText is only related to visual text, ignoring hidden elements and following styling rules, but it is a bit slow. textContent catches all lessons, including hidden elements, without formatting, it becomes quickly and more efficient.

Question-3: What is event delegation in the DOM?

The event delegation is a technique in the dom where an parent element handles events for its child elements using the event bubbling. Instead of adding events to many children's elements, we can connect the same listener with the parents and check which child has triggered the incident. It improves performance and works well for dynamic elements.

Question-4: What is event bubbling in the DOM?

The event bubbling is a behavior in the dom where an event begins with the target element and goes up through the ancestors.
